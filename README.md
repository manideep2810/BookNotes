# BookNotes

## To Run on Localhost
### 1. use command 'npm i' to install all node modules
### 2. Repalce the database Username and Password With you own Credentials
### 3. good to go use command nodemon index.js and check on http://localhost:3000/

## Features
### 1. Login/Signup Page
### 2. Home Page 
### 3. Search Books by Title,Author name,category
### 4. Save the books liked to My Collection Folder
### 5. Create and organize your Own custom Folders and add books to them appropriately
### 6. Sort the Books Based on Rating , Recency , and Author
### 7. You can save a Short Notes For Each books

## Technology and Tools Used 
### 1. HTML/CSS/JS for frontend
### 2. Bootstrap Jquery frontend frameworks
### 3. Nodejs Expressjs axios for Backend Logic
### 4. Open Library Cover API for Fetching Books detais
### 5. Database : Postgrees SQL
### 6. Bcrypt , Cookies , sessions For Authentication and Authorization
